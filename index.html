JavaScript: Break and Continue Statements

An overview of controlling loop execution in JavaScript.

Introduction

  • Control the flow of loops (for, while, do...while).
  • Useful for handling complex logic within loops.

The break Statement

  • Terminates the entire loop immediately.
  • Transfers control to the statement after the loop.
  • Often used when a specific condition is met.

Example:

for (let i = 1; i <= 10; i++) {
  if (i > 5) {
    break; // Exit the loop when i > 5
  }
  console.log(i); // Outputs: 1, 2, 3, 4, 5
}
console.log("Loop terminated.");

The continue Statement

  • Skips the remainder of the current loop iteration.
  • Immediately jumps to the next iteration.
  • Useful for bypassing code when specific conditions are met.

Example:

for (let i = 1; i <= 10; i++) {
  if (i % 2 === 0) {
    continue; // Skip even numbers
  }
  console.log(i); // Outputs: 1, 3, 5, 7, 9 (odd numbers)
}
console.log("Loop executed with continue.");

Summary

  • break exits the loop completely.
  • continue skips the current iteration and moves to the next.
  • Both help in writing efficient and clean loop logic.
